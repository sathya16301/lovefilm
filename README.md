# 🎀 lovefilm — Vintage Coquette Photobooth

lovefilm is a vintage coquette aesthetic photobooth web app that lets you capture moments with beautiful film strip layouts, retro filters, draggable stickers, doodle tools, and text overlays.

---

## 🌐 Live Demo
🔗 [sathya16301.github.io/lovefilm](https://sathya16301.github.io/lovefilm)

> ⚠️ Best experienced on desktop — mobile support coming soon!

---

## ✨ Features

- 📷 **Live Camera Feed** — Real-time webcam capture
- 🎞️ **4 Layouts** — Strip (3 frames), Grid (4 frames), Polaroid (1 frame), Duo (2 frames)
- 🎨 **8 Frame Themes** — Classic, Coquette, Vintage, Dreamy, Noir, Botanical, Glam, Sweet
- ✨ **5 Filters** — None, Vintage, Velvet, Bloom, Noir applied live to camera
- 🎀 **16 Stickers** — Draggable emoji stickers (double click to delete)
- ✏️ **Doodle Tool** — Draw on photos with 8 colours and 4 brush sizes
- 🔤 **Text Tool** — Add draggable text with 3 font choices
- ⏱️ **Timer** — 3 second countdown before capture
- 📸 **Shutter Animation** — Flash effect on capture
- 💾 **Download** — Save your complete photo strip as PNG
- 🖼️ **Gallery** — View and download all saved photos
